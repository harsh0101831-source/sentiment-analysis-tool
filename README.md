# sentiment-analysis-tool
Sentiment Analysis Tool using Python, Flask and TextBlob

## Overview
A sentiment analysis web application developed using Python, Flask, and the TextBlob library. The application analyzes user-input text and classifies it as Positive, Negative, or Neutral based on sentiment polarity.

## Features
- Analyze text entered by the user
- Predict Positive, Negative, or Neutral sentiment
- Polarity-based sentiment analysis using the TextBlob library
- Simple and interactive Flask web interface

## Tech Stack
- Python
- Flask
- TextBlob
- HTML
- CSS
